# Estudos Java - Programa Start (Capgemini) 🚀☕

Este repositório foi criado para documentar, armazenar e acompanhar a minha evolução no aprendizado da linguagem **Java** através do programa intensivo **Start da Capgemini**.

O objetivo principal aqui é o famoso "treino de dedos": transpor a minha base sólida de lógica de programação (desenvolvida em JavaScript) para a sintaxe e a arquitetura fortemente tipada do Java, criando memória muscular e dominando os fundamentos da linguagem.

---

## 🛠️ Tecnologias e Ferramentas Utilizadas

* **Linguagem Principal:** Java (JDK 17+)
* **IDE:** IntelliJ IDEA
* **Ambiente de Desenvolvimento:** WSL2 (Ubuntu)
* **Controle de Versão:** Git & GitHub

---

## 💻 Desafios Desenvolvidos Inicialmente

O repositório foi inaugurado com 3 mini-programas progressivos focados em conceitos essenciais de entrada, processamento e saída de dados via console:

### 1. Validador de Idade de Cinema 🎬
* **Conceitos aplicados:** Leitura de dados com `Scanner` (`nextLine`, `nextInt`), manipulação de tipos primitivos e estruturas condicionais controladas (`if/else`).

### 2. Calculador de Média Formatada 🧮
* **Conceitos aplicados:** Utilização do tipo de ponto flutuante `double`, cálculo aritmético e formatação avançada de strings com casas decimais fixas utilizando o método `System.out.printf()`.

### 3. Simulador de Inventário de RPG (Com Arrays) 🎮
* **Conceitos aplicados:** Laço de repetição (`while`), manipulação de tamanho de arrays estáticos em Java e a importância da ordem de incremento de variáveis de controle para prevenção de estouro de memória.
